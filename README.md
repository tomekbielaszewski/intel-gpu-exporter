# intel-gpu-exporter

Get metrics from Intel GPUs

## Deployment

Runs on port 8080. Wraps `intel_gpu_top` and exposes its output as Prometheus metrics.

### Docker Compose

```yaml
version: "3.8"

services:
  intel-gpu-exporter:
    image: ghcr.io/tomekbielaszewski/intel-gpu-exporter:latest
    container_name: intel-gpu-exporter
    restart: unless-stopped
    privileged: true
    pid: host
    ports:
      - 8080:8080
    volumes:
      - /dev/dri/:/dev/dri/
```
